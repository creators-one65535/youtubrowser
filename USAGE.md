# Youtubrowser 使い方マニュアル

対象バージョン: 0.5.0

## 1. 起動する

1. `Youtubrowser-0.5.0.zip` を任意のフォルダに展開します。
2. 展開したフォルダ内の `Youtubrowser.exe` を起動します。

zip の中から直接起動せず、必ず展開してから起動してください。

初回起動時は、既定のホームとして `https://www.youtube.com/` を開きます。

## 2. ツールバーを表示する

通常、上部のツールバーは非表示です。

マウスカーソルをアプリのウィンドウ上に乗せると、上部にHUD風のツールバーが表示されます。カーソルがツールバー上にある間は表示され続け、ウィンドウ外へ移動すると少し後に自動で隠れます。

ツールバーのボタンがない空きスペースは、ウィンドウ移動に使えます。

## 3. 基本操作

ツールバー左側のボタンで操作します。

- 戻る: 前のページへ戻ります。
- 進む: 次のページへ進みます。
- 再読み込み: 現在のページを再読み込みします。
- ホーム: 設定済みのホームURLを開きます。
- URL欄: URLまたは検索語句を入力します。
- 移動: URL欄の内容で移動します。

URL欄では、`youtube.com/watch...` のように `https://` を省略したURLも入力できます。URLとして解釈できない文字列は、Google検索として新しいタブで開きます。

## 4. ホーム画面を変更する

現在開いているページをホームにしたい場合は、次の手順で変更します。

1. ホームにしたいページを表示します。
2. ツールバーのホームボタンを右クリックします。
3. `現在のタブをホームに設定` を選びます。
4. 確認メッセージが出たら完了です。

次回からホームボタンや起動時の初期ページに、そのURLが使われます。

ホームURLはアプリ設定として保存されます。

## 5. タブを使う

- `+` ボタンで新しいタブを開きます。
- タブ名をクリックすると、そのタブに切り替わります。
- タブ右側の閉じるボタンでタブを閉じます。
- タブが増えて見切れる場合は、`<` / `>` ボタンでタブ列を横スクロールします。

最後のタブを閉じてもアプリは終了しません。かわりに、最近開いたページの履歴一覧タブが表示されます。

## 6. 左右分割表示

タブ内の右向きボタンを押すと、そのタブを右側ペインに固定表示できます。

左側で別のタブを操作しながら、右側に別ページを表示したままにできます。右側に表示したタブの左向きボタンを押すと、通常表示に戻ります。

## 7. 音声タブを操作する

音声を再生しているタブにはスピーカーアイコンが表示されます。

スピーカーアイコンをクリックすると、そのタブだけをミュートできます。もう一度クリックするとミュートを解除します。

## 8. 拡張機能を追加する

`ADD` ボタンから、アンパック済みのローカル拡張機能フォルダを追加できます。

1. 拡張機能をローカルフォルダへ展開します。
2. `manifest.json` が入っているフォルダを用意します。
3. ツールバーの `ADD` を押します。
4. そのフォルダを選択します。

Chromeウェブストアからのワンクリックインストールには対応していません。

## 9. 保存される情報

以下の情報は、ユーザーごとのローカル設定として保存されます。

- ウィンドウ位置
- ウィンドウサイズ
- 最大化状態
- ホームURL
- 表示履歴
- WebView2のCookieやログイン状態

主な保存先は `%LOCALAPPDATA%\Youtubrowser` です。

## 10. 困ったとき

### ツールバーが見えない

アプリのウィンドウ上にマウスカーソルを乗せて、少し待ってください。別のウィンドウが手前にある場合は表示されません。

### ホーム設定が変わらない

ホームにしたいページが現在のタブで表示されているか確認してください。`about:` など通常のWebページではないURLは、ホームとして保存できない場合があります。

### 起動しない

Windows 10以降またはWindows 11で実行してください。WebView2 Runtimeが入っていない環境では、Microsoft Edge WebView2 Runtimeのインストールが必要です。

### 設定を初期化したい

アプリを終了し、`%LOCALAPPDATA%\Youtubrowser` の設定ファイルやWebView2データを退避または削除すると、次回起動時に初期状態へ戻せます。
